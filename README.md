# WindowsPowerKits

WindowsPowerKits 是一个综合性的轻量 Windows 工具箱，兼容 Windows 10/11。

项目结构：
```
- functions             文件夹      工具目录
  system.bat            BAT脚本     系统工具部分
- scripts               文件夹      脚本目录
  op.bat                BAT脚本     格式化输出脚本
  wait.bat              BAT脚本     等待工具脚本
README.md               MD文档      说明文件（本文件）
WindowsPowerKits.bat    BAT脚本     工具主文件（入口）
```

主要使用：BAT 和 PowerShell 批处理脚本，执行效率高，编码简单。
